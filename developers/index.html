Developer Tools

Please find below a list of tools that may be useful in Web of Things (WoT) context.

Thing Description Tooling

  • Thing Description Playground (TD validation)
  • WoTify (a collection of devices that have been WoT-enabled)
  • Virtual Thing (creates and deploys a virtual thing based on its TD)
  • Web of Things Test Bench (tests a WoT Thing by executing interactions automatically, based on its Thing Description)

Scripting API Solutions

  • thingweb node-wot (Eclipse Public License & W3C Software License)
    • Browsified node-wot (Web UI)
  • WoT FXUI (UI for desktop, mobile, browser)
    • See running Web-UI instance
